Watch it work

Real artifacts from the demo audit — the query that catches the crime, the sandbox that proves it, the verdict that lands in the graph.

① The detector — Cypher on Neo4j Aura
MATCH (t:Cell {kind:'formula'})
WHERE t.formula CONTAINS 'SUM('
MATCH (p:Cell)-[:FEEDS_INTO]->(t)
WITH t, collect(p) AS ps, max(p.row) AS lastRow
MATCH (gap:Cell {col:t.col})
WHERE gap.row > lastRow AND gap.row < t.row
  AND NOT (gap)-[:FEEDS_INTO]->(t)
RETURN t, gap   → 12 totals stop short

Why it's different

Spreadsheet linters flag patterns and stop. SheetSleuth closes the loop.

🕸 It sees the wiring A workbook is thousands of invisible "this feeds that" wires. We rebuild them as a Neo4j graph, so questions like "what depends on this cell?" become one traversal — not an afternoon of clicking Trace Precedents.
🧪 It proves, not lints Every suspect gets executed: a fresh Daytona sandbox recomputes your entire workbook with and without the fix and measures exactly which numbers change, in dollars. Verdicts, not vibes.
🧠 It answers with receipts Ask "what if payroll rises 10%?" — the cloud agent won't guess. It runs the scenario, cites the measured before/after, and remembers your past audits with long-term memory.

The loop, live

Not a linter. A closed loop: extract → graph → detect → execute → measure → remember.

1Parse Every formula reference becomes a FEEDS_INTO edge in a Neo4j Aura property graph.
2Detect Cypher traversals find structural smells; an LLM triages the semantic ones.
3Prove A fresh Daytona sandbox recomputes the entire workbook, as-is vs fixed, and diffs every cell.
4Ask A RocketRide Cloud agent answers what-ifs by actually running them — and remembers past audits with Cognee.
